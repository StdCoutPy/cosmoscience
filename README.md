# 🌌 CosmoScience — Интерактивный портал о науке

Профессиональное веб-приложение, объединяющее физику, математику, информатику и астрономию в едином интерактивном интерфейсе. 

**[🚀 Посмотреть проект вживую](https://stdcoutpy.github.io/cosmoscience/index.html)**

## ✨ Ключевые особенности
*   **Creative UI:** Стеклянный дизайн (Glassmorphism) с анимированными неоновыми эффектами.
*   **Canvas Engine:** Интерактивное созвездие навигации с динамическим расчетом координат.
*   **Procedural Audio:** Генерация космического амбиента и звуковых откликов через Web Audio API (без внешних аудиофайлов).
*   **Simulations:** 
    *   Гравитационная линза (Черная дыра) за курсором.
    *   Генератор уникальных экзопланет на основе случайных величин.
    *   Физические маятники и алгоритмы сортировки.

## 🛠 Технологии
*   **Core:** HTML5, CSS3 (Flexbox, Grid, Clamp)
*   **Logic:** Pure JavaScript (Vanilla JS)
*   **Graphics:** Canvas API (2D Context)
*   **Audio:** Web Audio API (Oscillator nodes, LFO)

## 📦 Как запустить локально
1. Склонируйте репозиторий:
   `git clone https://github.com`
2. Откройте `index.html` в любом современном браузере.
